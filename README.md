# Linux Practice Notes

This repository contains short, focused notes for practicing basic Linux commands.

  File and folder commands, path, writing/editing files, copy/move, delete, curl, memory usage.

- [docs/linux-network-basics.md](docs/linux-network-basics.md)  
  IP address, open ports, hostname, DNS, download files, `/etc/hosts`, change SSH port.

- [docs/linux-package-management.md](docs/linux-package-management.md)  
  Package managers (`apt`, `yum`, `dnf`), upgrade all packages, install/remove `nginx`.

- [docs/linux-log-and-system-info.md](docs/linux-log-and-system-info.md)  
  SSH logs, system time, CUBE RAM disk usage, DNS lookup (`dig`, `nslookup`, `host`), `traceroute`.
